• add(value)

    インスタンス毎に一意になる連番をキーとしてvalueを保存します。連番が返り値になります。

    collection.add({
    cir: 'Cool is Right.'
});

  • each(func)

    保存されたデータごとにfuncを実行します。

    collection.each(function(val, key, obj) {
    // val: setなどで保存したデータ
    // key: set時に指定したキー名
    // obj: 保存データすべてを持ったオブジェクト
});

  • fire(key, [value])

    onオプション、onメソッドで登録されたイベントを実行します。

    model.on('change', function(vars) {
    // write code.
    console.log(vars); // {cir: 'Cool is Right.'}
});

observer.fire('change', {
    cir: 'Cool is Right.'
});

  • get([key])

    データを取り出します。

    model.get('test');

  • map(func)

    保存されたデータごとにfuncを実行します。funcの返り値が対応する保存されていたデータを上書きします。

    collection.map(function(val, key, obj) {
    // val: setなどで保存したデータ
    // key: set時に指定したキー名
    // obj: 保存データすべてを持ったオブジェクト

    return val; // returnの値で上書き
});

  • off(key, func)

    登録されたイベントを削除します。

    var func1 = function() {
    console.log('func1');
};
var func2 = function() {
    console.log('func2');
};

model.on('change', func1);
model.on('change', func2);

model.off('change', func1);
model.fire('change'); // exe func2.

  • on(key, func)

    イベントを登録します。
    登録されたイベントはfireメソッドで強制的に呼び出しできます。

    var func1 = function() {
    console.log('func1');
};
var func2 = function() {
    console.log('func2');
};

model.on('change', func1);
model.on('change', func2);

model.off('change', func1);
model.fire('change'); // exe func2.

  • prev([key])

    保持しているデータの一つ前の状態を返します。
    keyを省略した場合、データを纏めたオブジェクトを返します。
    changeイベントが発火した場合に前の数値を使用したい場合などに便利です。

    model.set('test', 12345);
model.set('test', 23456);
model.prev('test'); // 12345

  • remove(key)

    データを削除します。
    このメソッドを実行した場合、onオプション,もしくはonメソッドで登録したイベントを実行します。

    model.remove('test'); // removeイベント実行

  • reset(key)

    保存された値を全て削除します。
    このメソッドを実行した場合、onオプション,もしくはonメソッドで登録したイベントを実行します。

    model.reset(); // resetイベント実行

  • set(key, value)

    Ollectionにデータを保存します。
    このメソッドを実行した場合、onオプション,もしくはonメソッドで登録したイベントを実行します。

    model.set('test', 12345); // changeイベント実行
