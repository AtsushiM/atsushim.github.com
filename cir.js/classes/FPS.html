• enter()

    間隔実行される処理を強制的に発火します。

    fps.enter();

  • getCriterion()

    初期化時に指定したFPS目標値(criterion)を取得します。

    var criterion = fps.getCriterion(); // Number

  • getFrameTime()

    FPS目標値の1フレームあたりのミリ秒を取得します。

    var frametime = fps.getFrameTime(); // Number

  • getSurver()

    現在のFPSを取得します。

    var surver = fps.getSurver(); // Number

  • start()

    間隔実行を開始します。
    インスタンス生成時に自動的に呼び出されます。
    呼び出しをしたくない場合、manualオプションをtrueにしてください。

    fps.start();

  • stop()

    間隔実行を停止します。

    fps.stop();
