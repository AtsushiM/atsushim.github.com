• start()

    リクエストを開始します。
    インスタンス生成時に自動的に呼び出されます。
    呼び出しをしたくない場合、manualオプションをtrueにしてください。
    startイベントを呼び出します。

    ajax.on('start', function() {
    // write code.
});
ajax.start();

  • stop()

    リクエストを停止します。
    stopイベントを呼び出します。

    ajax.on('stop', function() {
    // write code.
});
ajax.stop();
