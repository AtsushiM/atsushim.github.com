• getCurrent()

    現在の秒数を取得します。

    var currenttime = movie.getCurrent(); // Number

  • getDuration()

    音声ファイルの総再生時間を取得します。

    var duration = movie.getDuration(); // Number

  • getElement()

    インスタンス生成時に作られたVideo要素を取得します。

    var video = movie.getElement(); // element

  • loop(bool)

    ループの設定を行います。

    movie.loop(true); // ループ有り
movie.loop(false); // ループ無し

  • pause()

    一時停止します。

    movie.pause();

  • play()

    再生します。

    movie.play();

  • setCurrent()

    音声ファイルの再生位置を設定します。

    movie.setCurrent(1); // 1秒の位置へ

  • stop()

    停止します。
    再生位置は0秒になります。

    movie.stop(); // 停止して0秒の位置へ
