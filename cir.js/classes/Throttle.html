• lock()

    処理リクエストを受け付けなくします。

    throttle.lock();
throttle.request('Cool is Right.'); // 実行されない

  • request([vars])

    処理をリクエストします。
    waittimeオプションだけ連続実行をできなくします。

    throttle.request('Cool is Right.');
throttle.request('Hot is Right.'); // 連続実行出来ない

  • unlock()

    処理リクエストを受け付けるようにします。

    throttle.lock();
throttle.unlock();
throttle.request('Cool is Right.'); // 実行される
