• addition(num1, num2)

    num1にnum2を加算します。

    var num = calc.addition(0.1, 0.2); // 0.3

  • division(num1, num2)

    num1にnum2を除算します。

    var num = calc.division(0.3, 0.1); // 3

  • multiplication(num1, num2)

    num1にnum2を乗算します。

    var num = calc.multiplication(0.1, 0.1); // 0.01

  • subtraction(num1, num2)

    num1にnum2を減算します。

    var num = calc.subtraction(1, 0.9); // 0.1
