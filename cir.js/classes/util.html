• abstractFunction()

    呼び出すとエラーになるメソッドです。
    C.lassなどでクラス継承等を行う際、実装を強制したいメソッドに指定するなどの使い方ができます。

    var SuperClass = C.lass({
    method: util.abstractFunction
});
var Class = SuperClass.extend();

var instance = new Class();

instance.method(); // Classでmethodを実装していないため、エラー表示

  • binarySearch({low: number, high: number, compare: function, end: function})

    二分探索のアルゴリズムを提供するメソッドです。
    開始する最低値、最高値と比較用、探索終了時用の関数を渡すことで二分探索を行います。
    compareには探索方向が低くなる場合にtrueを返す関数を指定します。

    var ary = [],
    i = 0,
    len = 100,
    answer;

for (; i < len; i++) {
    ary[i] = i;
}

util.binarySearch({
    low: 0,
    high: 100,
    compare: function(point) {
        if (point < 50) {
            return true;
        }
    },
    end: function(point) {
        answer = point;
    }
});

  • checkUserAgent(regex, [userAgent])

    userAgentに対してregexで正規表現を行い、結果をBooleanで返します。
    userAgentが省略された場合、window.navigator.userAgentに対して正規表現でチェックを行います。

    var check = util.checkUserAgent(/iPhone|iPad|iPod/i); // boolean

var copyobj = C.util.copyObj(orgobj); // orgobj !== copyobj

  • nullFunction()

    何もしないメソッドです。
    メソッドの引数で渡すcallbackが必須でない場合等で使うと効果的です。

  • override(targetObj, varObj)

    targetObjに対してvarObjのプロパティを上書きします。
    返り値は上書きされたtargetObjになります。

    var target = {};
var over = {sample: 'override property'};
util.override(target, over);
alert(target.sample); // override property

  • owner(ownerObj, [methods], [overrideObj])

    methods内のすべてのメソッドのthisをownerObjに固定し、上書きを行います。
    overrideObjが指定された場合、overrideObjを新しいメソッドで上書きし、その後ownerObjに適用されます。
    methodsが省略された場合、methodsはownerObjになります。
    overrideObjが省略された場合、overrideObjはmethodsになります。
    クラスのメソッドのthisを固定したい場合などに便利です。

    var Test = C.lass.extend({
    init: function(config) {
        util.owner(this, this, config);
    },
    no: 0,
    test: 'Override Test.',
    countup: function() {
        this.no++;
        console.log(this.no);
    }
});
var test = new Test({
    test: 'Cool is Right.'
});
console.log(test.test); // 'Cool is Right.'
setInterval(test.countup, 1000);
