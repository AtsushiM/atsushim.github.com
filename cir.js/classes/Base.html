• dispose()

    インスタンスが使用しているメモリを解放します。
    インスタンスのプロパティとメソッドを全て削除します。
    メモリが解放されるタイミングはブラウザのガベージコレクションに依存します。
    Baseを継承したクラスでdisposeメソッドに何か処理を追加したい場合、disposeInternalメソッドを作成すればdispose実行時に呼び出されます。
    例えばsetIntervalなどのタイマーや追加したエレメントの削除などが考えられます。

    base.dispose();

  • contract(element, eventname, handler)

    element.addEventListnerを実行します。
    このメソッドは紐付けた引数と一意になる値を返します。
    この返り値をuncontractメソッドに渡せば登録したイベントを削除します。
    このメソッドで登録したイベントはdisposeメソッド実行時に自動的に削除されます。

    var $sample = el.$('#sample');
var contractid = base.contract($sample, C.e.CLICK, function() {
    alert('sample click!');
}); // イベント登録
base.uncontract(contractid); // イベント削除

  • uncontract(id)

    contractメソッドの返り値を元にelement.removeEventListnerを実行します。

    var $sample = el.$('#sample');
var contractid = base.contract($sample, C.e.CLICK, function() {
    alert('sample click!');
}); // イベント登録
base.uncontract(contractid); // イベント削除
