  • SWITCHCLICK

    実行している環境がタッチパネル対応の場合、'touchstart'に、
    タッチパネル非対応の場合、'click'になるプロパティです。

    var $sample = C.$('#sample');
$sample.on(C.e.SWITCHCLICK, function() {
    // write code.
});
