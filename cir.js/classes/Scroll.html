• kill()

    ブラウザスクロールを不可にします。

    scroll.kill(); // スクロールを停止

  • revival()

    ブラウザスクロールを可能にします。

    scroll.revival(); // スクロールを停止

  • scrollY()

    現在のY軸のスクロール位置を取得します。

    scroll.scrollY(); // Number

  • smooth(target, callback)

    スムーズスクロールで移動します。
    targetには数値かHTMLエレメントが指定出来ます。

    scroll.smooth(0); // ページトップへ
scroll.smooth(C.dom.$('#test'), function() {
    // write code.
}); // #testの位置へ

  • to(num)

    スクロール位置を移動します。

    scroll.to(100); // トップから100pの位置へ

  • toBottom()

    スクロール位置を一番下にします。

    scroll.toBottom();

  • toTop()

    スクロール位置を一番上にします。

    scroll.toTop();
