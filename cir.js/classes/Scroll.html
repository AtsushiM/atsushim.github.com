• checkBottom([offset])

    スクロールがページ下部まで到達しているかbooleanで返します。offsetに数値が渡された場合、ページ下部から数値の範囲にスクロール位置があればtrueを返します。

    scroll.checkBottom(); // boolean

  • kill()

    ブラウザスクロールを不可にします。

    scroll.kill(); // スクロールを停止

  • percent([{low: number, high: number, now: number}])

    現在のスクロール位置を0~1で返します。low,high,nowを渡した場合、それらを使用して値を計算します。

    scroll.percent(); // ページ全体のスクロール位置

  • revival()

    ブラウザスクロールを可能にします。

    scroll.revival(); // スクロールを停止

  • scrollLimit()

    scrollTop()で返される値の限界を取得します。

    scroll.scrollTop(); // Number

  • scrollTop()

    現在のY軸のスクロール位置を取得します。

    scroll.scrollTop(); // Number

  • smooth(target, callback)

    スムーズスクロールで移動します。
    targetには数値かHTMLエレメントが指定出来ます。

    scroll.smooth(0); // ページトップへ
scroll.smooth(C.dom.$('#test'), function() {
    // write code.
}); // #testの位置へ

  • to(num)

    スクロール位置を移動します。

    scroll.to(100); // トップから100pの位置へ

  • toBottom()

    スクロール位置を一番下にします。

    scroll.toBottom();

  • toTop()

    スクロール位置を一番上にします。

    scroll.toTop();
