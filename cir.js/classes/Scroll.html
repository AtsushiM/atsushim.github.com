• getLimit()

    getTop()で返される値の限界を取得します。

    scroll.getLimit(); // Number

  • getRatio([{low: number, high: number, now: number}])

    現在のスクロール位置を0~1で返します。low,high,nowを渡した場合、それらを使用して値を計算します。

    scroll.getRatio(); // ページ全体のスクロール位置

  • getY()

    現在のY軸のスクロール位置を取得します。

    scroll.getY(); // Number

  • isBottom([offset])

    スクロールがページ下部まで到達しているかbooleanで返します。offsetに数値が渡された場合、ページ下部から数値の範囲にスクロール位置があればtrueを返します。

    scroll.isBottom(); // boolean

  • kill()

    ユーザー操作によるブラウザスクロールを不可にします。

    scroll.kill(); // スクロールを停止

  • revival()

    ユーザー操作によるブラウザスクロールを可能にします。

    scroll.revival(); // スクロールを停止

  • to(num)

    スクロール位置を移動します。

    scroll.to(100); // トップから100pの位置へ

  • toBottom()

    スクロール位置を一番下にします。

    scroll.toBottom();

  • toSmooth(target, callback)

    スムーズスクロールで移動します。
    targetには数値かHTMLエレメントが指定出来ます。

    scroll.toSmooth(0); // ページトップへ
scroll.toSmooth(C.dom.$('#test'), function() {
    // write code.
}); // #testの位置へ

  • toTop()

    スクロール位置を一番上にします。

    scroll.toTop();
