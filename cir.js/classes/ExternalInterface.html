• addCallback(name, function)

    ウェブビューからネイティブにメソッドを提供します。
    iOSの場合はlocation.hashに#メソッド名?変数1=値&変数2=値...
    Androidの場合も同様にメソッドに'メソッド名?変数1=値&変数2=値...' を引数として渡せばメソッドを呼び出します。

    external.addCallback('webviewMethod', function(vars) {
    // write code.
    console.log(vars); // {text: 'Coll is Right.'}
});

// iOSモードの場合
location.hash = 'webviewMethod?text=Cool%20is%20Right.';

  • call({mode: string, vars: {key: val}})

    ネイティブ側から提供されたメソッドを実行します。
    iOSの場合はlocation.hashに特定の形式の文字列を設定します。
    Androidの場合、インスタンス生成時に渡したオブジェクト内のメソッドを呼び出します。

    external.call({
    mode: 'nativeMethod', // ネイティブ側の呼び出す機能名
    vars: {
        str: 'Cool is Right.',
        num: 123
    }
});
// iOSの場合、location.hashに [ #nativeMethod?str=%22Cool%20is%20Right.%22&num=123 ]
// Androidの場合、関数の引数に"nativeMethod?str=%22Cool%20is%20Right.%22&num=123"

  • removeCallback(name)

    addCallbackで登録したメソッドを削除します。

    external.addCallback('webviewMethod', function(vars) {
    // write code.
    console.log(vars); // {text: 'Coll is Right.'}
}); // 登録
external.removeCallback('webviewMethod'); // 削除

// iOSモードの場合
location.hash = 'webviewMethod?text=Cool%20is%20Right.'; // 削除しているため呼び出されない
