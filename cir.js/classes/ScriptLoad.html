• request({src, [callback]})

    jsファイルの動的読み込みを行います。

    scriptload.request({
    src: '/js/dynamic.js',
    callback: function() {
        // write code.
    }
});

  • requests([{src, [callback]}], [callback])

    複数ファイルの動的読み込みを行います。

    scriptload.requests([
    { src: '/js/c.js', callback: function() { /* write code. */}},
    { src: '/js/i.js', callback: function() { /* write code. */}},
    { src: '/js/r.js', callback: function() { /* write code. */}}
], function(scripts) {
    // callback.
});
