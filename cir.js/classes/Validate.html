• displayError(key, text)

    levelオプションに合わせてエラー表示を行います。

    validate('test', 'Test'); // Validate Error: test is Test.

  • isArray(key, vars)

    varsがArrayの場合はtrueを返し、そうでない場合エラー表示します。

    var check;
check = validate.isArray('check', []); // true
check = validate.isArray('check', {}); // error

  • isBoolean(key, vars)

  • isFunction(key, vars)
