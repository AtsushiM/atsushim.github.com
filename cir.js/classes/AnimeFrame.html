• request(callback)

    callbackをアニメーションするタイミングで実行します。タイミングはブラウザ依存です。
    ブラウザがrequestAnimationFrameに対応していない場合、自動的にsetTimeoutを代用します。

    frame.request(function(){
    // write code.
});

  • cancel(id)

    requestメソッドの返り値を引数に、アニメーションの実行をキャンセルします。

    var id = frame.request(function(){
    // write code.
});
frame.cancel(id);
