• getLimitFontSize(text)

    textをel.innerHTMLに代入した場合、width,heightの値を超えない限界のフォントサイズを返します。8px以下になった場合、数値は0になります。

    limittext.getLimitFontSize('Cool is Right.'); // Number

  • getLimitTextLength(text)

    textをel.innerHTMLに代入した場合、width,heightの値を超えない限界のテキスト長を返します。

    limittext.getLimitTextLength('Cool is Right.'); // Number
