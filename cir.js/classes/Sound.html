• getElement()

    インスタンス生成時に作られたAudio要素を取得します。

    var audio = sound.getElement(); // element

  • getCurrent()

    現在の秒数を取得します。

    var currenttime = sound.getCurrent(); // Number

  • getDuration()

    音声ファイルの総再生時間を取得します。

    var duration = sound.getDuration(); // Number

  • loop(bool)

    ループの設定を行います。

    sound.loop(true); // ループ有り
sound.loop(false); // ループ無し

  • pause()

    一時停止します。

    sound.pause();

  • play()

    再生します。

    sound.play();

  • setCurrent()

    音声ファイルの再生位置を設定します。

    sound.setCurrent(1); // 1秒の位置へ

  • stop()

    停止します。
    再生位置は0秒になります。

    sound.stop(); // 停止して0秒の位置へ
