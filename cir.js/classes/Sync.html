• addTask(task, [ priority ])

    タスクを追加します。priorityが指定されなかった場合、キューの最後に追加されます。priorityは整数で指定し、0に近いほど優先度が上がります。
    changeイベントを呼び出します。

    sync.on('change', function() {
    // write code.
});
sync.addQueue(function() {
    // write code.
}, 0); // 最優先でタスクを登録

  • getQueue(queue)

    キューを取得します。

    sync.getQueue(); // Array

  • pause()

    キューの処理を一時停止します。
    pasuseイベントを呼び出します。

    sync.on('pause', function() {
    // write code.
});
sync.pause();

  • setQueue(queue)

    キューを書き換えます。
    changeイベントを呼び出します。

    sync.on('change', function() {
    // write code.
});
sync.setQueue([function() {
    // write code.
}]);

  • start()

    タスクを開始します。
    startイベントを呼び出します。

    sync.on('start', function() {
    // write code.
});
sync.start();

  • stop()

    タスクを停止します。
    キューは空になります。
    stopイベントを呼び出します。

    sync.on('stop', function() {
    // write code.
});
sync.stop();

  • removeTask(task)

    タスクを削除します。
    changeイベントを呼び出します。

    var action = function() {
    // write code.
};
sync.addTask(action);

sync.on('change', function() {
    // write code.
});

sync.removeTask(action);

  • resetQueue([ queue ])

    キューをインスタンス生成時の状態にリセットします。queueが渡された場合、その値を初期値に設定します。
    resetイベントを呼び出します。

    sync.on('reset', function() {
    // write code.
});
sync.resetQueue([function() {
    // write task.
}]);

  • restart([ queue ])

    キューをインスタンス生成時の状態にリセットします。queueが渡された場合、その値を初期値に設定します。
    その後startメソッドを実行します。

    sync.on('reset', function() {
    // write code.
});
sync.on('start', function() {
    // write code.
});
sync.restart();

  • resume()

    キューの処理を再開します。
    resumeイベントを呼び出します。

    sync.on('resume', function() {
    // write code.
});
sync.resume();
