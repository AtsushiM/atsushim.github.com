• start()

    タスクを開始します。

    task.start();

  • resetQueue([ queue ])

    キューをインスタンス生成時の状態にリセットします。queueが渡された場合、その値を初期値に設定します。

    task.resetQueue([function() {
    // write code.
}]);

  • setQueue(queue)

    キューを書き換えます。

    task.setQueue([function() {
    // write code.
}]);

  • getQueue(queue)

    キューを取得します。

    task.getQueue(); // Array

  • addTask(task, [ priority ])

    タスクを追加します。priorityが指定されなかった場合、キューの最後に追加されます。priorityは整数で指定し、0に近いほど優先度が上がります。

    task.addQueue(function() {
    // write code.
}, 0); // 最優先でタスクを登録

  • removeTask(task)

    タスクを削除します。

    var action = function() {
    // write code.
};
task.addTask(action);
task.removeTask(action);
