• acceptRanges()

    サーバーからAccept-Rangesを取得します。

    var acceptRanges = server.acceptRanges();

  • connection()

    サーバーからConnectionを取得します。

    var connection = server.connection();

  • contentLength()

    サーバーからContent-Lengthを取得します。

    var contentLength = server.ContentLength();

  • contentType()

    サーバーからContent-Typeを取得します。

    var contentType = server.contentType();

  • date(callback)

    サーバから日付を取得します。
    このメソッドのみ、非同期で実行されるためcallbackが必要です

    server.date(function(date) {
    // write code.
    console.log(date);
});

  • keepAlive()

    サーバーからKeep-Aliveを取得します。

    var keepAlive = server.keepAlive();

  • lastModified()

    サーバーからLast-Modifiedを取得します。

    var lastModified = server.lastModified();

  • server()

    サーバーからServerを取得します。

    var _server = server.server();
