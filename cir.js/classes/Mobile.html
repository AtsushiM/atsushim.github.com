• getZoom()

    画面の拡大率を取得します。

    var zoom = mobile.getZoom(); // zoom > 0

  • hideAddress()

    アドレスバーを非表示にします。

    mobile.hideAddress();

  • isAndroid()

    端末OSがAndroidかどうかBooleanで返します。

    var check = mobile.isAndroid(); // boolean

  • isFBAPP()

    Facebookアプリで閲覧してるかどうかBooleanで返します。

    var check = mobile.isFBAPP(); // boolean

  • isIOS()

    端末OSがiOSかどうかBooleanで返します。

    var check = mobile.isIOS(); // boolean

  • isMobile()

    モバイル端末かどうかBooleanで返します。

    var check = mobile.isMobile(); // boolean

  • isWindows()

    端末OSがWindowsかどうかBooleanで返します。

    var check = mobile.isWindows(); // boolean
