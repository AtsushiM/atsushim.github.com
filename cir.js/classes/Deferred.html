• done(function)

    遅延処理を登録します。

    deferred.done(function() {
    // write code.
})
.done(function() {
    // write code.
});

deferre.resolve(); // まとめて実行

deferred.done(function() {
    // write code.
}); // resolve()後は即座に実行

  • isResolve()

    遅延処理が実行済みかどうかBooleanで返します。

    if (!deferred.isResolve()) {
    alert('No Resolve.');
}
deferred.resolve();
if (deferred.isResolve()) {
    alert('Resolve.');
}

  • resolve()

    登録された遅延処理を実行します。

    deferred.done(function() {
    // write code.
})
.done(function() {
    // write code.
});

deferre.resolve(); // まとめて実行

deferred.done(function() {
    // write code.
}); // resolve()後は即座に実行
