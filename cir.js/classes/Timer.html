• countDown()

    カウントダウンを開始します。

    timer.countDown();

  • getLimit()

    リミット(秒)を取得します。

    var limit = timer.getLimit();

  • getProgress()

    経過%を0~1の間の数値で返します。

    var progress = timer.getProgress();

  • getTime()

    templateオプションに合わせた現在秒数を取得します。

    var time = timer.getTime();

  • setTimeup(function)

    リミットに到達した場合に実行されるontimeupオプションを設定します。

    timer.setTimeup(function(time) {
    // write code.
    console.log(time);
});

  • setUpdate(function)

    intervalオプションごとに実行されるonupdateオプションを設定します。

    timer.setUpdate(function(time) {
    // write code.
    console.log(time);
});

  • stop()

    タイマーを停止します。

    timer.stop();
